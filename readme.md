## 概要

このコードは、家の Wi-Fi ルータが不安定になるため、
**再起動を自動化する目的**で作成したものです。

本来は **iPhone（Scriptable）での使用**を想定して実装しました。

## 背景

家の Wi-Fi ルータが古く、定期的に再起動しないと通信が不安定になるため、
ブラウザから再起動ページへアクセスする操作を自動化しようとしました。

## 実装内容

- ルータの再起動ページ URL へのアクセス
- 再起動用 POST リクエストの送信
- Scriptable 上で JavaScript を実行する構成

## 結果と課題

iPhone の仕様上、**JavaScript を手動操作なしで実行することが難しく**、
Scriptable でもユーザー操作が必須となりました。

そのため、

> 自動化したつもりでも、結局は手動で再起動するのと変わらない

という結論に至り、本実装は没としました。

## 学び

iPhone（iOS）では、
- バックグラウンド実行の制限
- 自動タップやページ操作の制約

が非常に厳しく、  
**「JavaScriptが動く = 自動化できる」ではない**という点を学びました。

仕様調査をより丁寧に行ってから実装すべきだったと反省しています。

 node RestartWi-Fi.js
